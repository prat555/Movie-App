# Movie App

A React Native movie app built with Expo.

## Features

- Browse movies with mock data
- Movie details with cast and crew information
- Download and like functionality
- Modern UI with dark theme
- Responsive design

## Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Start the App

```bash
npm start
```
